# Tic Tac Toe
A browser-based Tic Tac Toe game built with HTML, CSS, and JavaScript.

## Features
- Player vs Player mode
- Player vs Computer mode (random AI)
- Score tracking
- Winning cell highlight

## How to Play
1. Open `TicTacToe.html` in your browser
2. Click a cell to make your move
3. First to get 3 in a row wins!
4. Click anywhere to restart after a game ends

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript
